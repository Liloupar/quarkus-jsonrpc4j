== Installation

If you want to use this extension, you need to add the `io.quarkiverse.jsonrpc:quarkus-jsonrpc` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>com.github.zohar-soul.quarkus-jsonrpc4j</groupId>
    <artifactId>quarkus-jsonrpc</artifactId>
</dependency>
----
